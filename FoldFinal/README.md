# FoldCounter - 折叠屏翻折计数器

一款专为折叠屏手机设计的翻折次数统计 APP，通过铰链角度传感器精准检测每一次开合动作，后台常驻运行，全程不漏计。

---

## 功能特性

- 实时统计每次合屏动作，数字即时更新
- 前台服务常驻后台，关闭 APP 仍持续计数
- 通知栏实时显示当前次数，无需打开 APP
- 本次会话 / 累计次数双重统计，数据持久保存
- 手机重启后自动恢复运行
- 支持单独重置本次或清除全部历史记录

## 如何编译安装

### 第一步：创建 GitHub 仓库

1. 访问 [github.com](https://github.com)，登录账号
2. 点击右上角 **+** → **New repository**
3. 填写仓库名（如 `FoldCounter`），选择 **Public**，点击 **Create repository**

### 第二步：上传项目文件

1. 进入新建的仓库页面
2. 点击 **Add file** → **Upload files**
3. 将本压缩包解压，把 `FoldFinal/` 文件夹内的**所有文件和文件夹**拖入上传区域
4. 点击 **Commit changes**

### 第三步：等待自动编译

1. 点击仓库顶部 **Actions** 标签页
2. 看到 **Build APK** 任务正在运行（黄色圆圈）
3. 等待约 3～5 分钟，变为绿色 ✅ 即编译成功

### 第四步：下载 APK

1. 点击绿色的 **Build APK** 记录
2. 页面底部 **Artifacts** 区域，点击 **FoldCounter-debug** 下载
3. 解压得到 `app-debug.apk`

### 第五步：安装到手机

1. 开启手机"允许安装未知来源应用"
   - 设置 → 安全 → 安装未知应用 → 允许
2. 将 APK 传输到手机（微信、数据线均可）
3. 点击 APK 文件，按提示安装

## 使用说明

| 操作 | 说明 |
|------|------|
| 合上屏幕 | 计为一次翻折，次数 +1 |
| 展开屏幕 | 状态重置，等待下次合屏 |
| Reset Session | 清零本次会话计数 |
| Clear Total | 清除所有历史累计数据 |
| Stop Service | 结束后台服务，停止统计 |

---

## 注意事项

- 需要**折叠屏手机**，普通手机无铰链传感器，APP 会提示不支持
- 系统要求 **Android 11（API 30）及以上**
- 国产 ROM（小米/华为/OPPO 等）请在设置中将本 APP 的后台权限设为**无限制**，防止被系统杀掉
  - 路径：设置 → 应用 → FoldCounter → 电池 → 无限制
